# Chrome Extension Setup

1. Open Chrome and go to `chrome://extensions`.
2. Enable Developer mode.
3. Click Load unpacked.
4. Select this folder:
   - `/Users/kalk3y/Desktop/tools/ytlbykalki/chrome_extension`
5. Pin the extension.
6. Make sure you are signed into YouTube in the same Chrome profile.
7. Open a channel About page.
8. Open extension popup, paste 2Captcha key, click Save Key.
9. Click Run On Current Channel.

The extension runs directly inside your signed-in browser session, so it avoids the guest-profile Selenium login problem.
